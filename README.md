# Hospital Personnel System

## 📌 Project Description

This project is a database system designed to manage hospital personnel, including departments, roles, shifts, qualifications, and attendance records.

## ⚙️ Technologies Used

* SQL Server
* Relational Database Design

## 📂 Project Structure

* `sql/` → Contains all SQL scripts (database, tables, queries)
* `report/` → Final project report
* `video/` → Presentation video (to be uploaded)

## ▶️ How to Run

1. Open SQL Server Management Studio
2. Run the SQL scripts in this order:

   * Create database
   * Create tables
   * Insert data
   * Run queries
